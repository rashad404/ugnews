1) MilliÖN, e-Manat Aparatlarından Paymanat Kod almaq üçün Ödəmə kartları menyusuna daxil olunur, Paymanat Code düyməsi tapılıb vurulur, Açılan ödəniş menyusunda kartın şifrə və seriya nömrəsinin SMS vasitəsilə mobil telefona göndərilməsindən ötrü mütləq olaraq işlətdiyiniz hər hansı mobil operatora məxsus telefon nomrə lazımi xanaya qeyd olunur, 1 AZN-dən kiçik olmayan istənilən məbləğ aparata daxil edilir və təsdiqləmə düyməsi vurulur.
